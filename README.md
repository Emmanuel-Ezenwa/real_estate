# Nigeria Real Estate Market Intelligence & Price Prediction

## 📊 Project Overview
This project analyses Nigeria’s real estate and construction market using data science techniques.

The goal is to:
- Understand market trends
- Identify high-growth cities
- Segment real estate developments
- Predict property values using machine learning

---

## 🧠 Key Insights

- Lagos dominates construction activity and total market value
- Market demand shows steady growth over time
- Real estate developments can be grouped into 3 clusters:
  - Low-value housing
  - Mid-tier developments
  - High-end luxury projects
- Location and category are the strongest drivers of property value

---

## 📈 Analysis Performed

- Exploratory Data Analysis (EDA)
- Market Share by City
- Time Series Trend Analysis
- Demand Forecasting
- Clustering (KMeans)
- Machine Learning (Random Forest Regression)

---

## 🤖 Machine Learning Model

Model used: Random Forest Regressor

Target:
- Property / Project Value

Features:
- City
- Category
- Time (month, year)
